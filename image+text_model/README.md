# Image+Text Model

This folder contains the code and resources for the **Image+Text multimodal model** within the Multimodal Medical Foundation Model project.

## Overview

- Implements a transformer-based architecture that merges visual and textual inputs.
- Designed for medical imaging tasks combined with textual reports.
- Includes notebooks for training and evaluation (not yet pushed).

## Directory Structure

- `multimodal_medical_foundation_model.ipynb` – Main notebook for model development.
- `check_nb.py` – Utility script to verify notebook execution.

*Further content will be added as the model matures.*
