# Multimodal Medical Foundation Model

## Overview

This repository hosts a **Multimodal Medical Foundation Model** that focuses on image‑only processing for medical imaging tasks. The project provides a flexible framework for training, evaluating, and deploying deep learning models on radiology datasets.

cd multimodal-medical-foundation-model

## Usage

1. **Explore notebooks** – The `image_only` folder contains ready‑to‑run Jupyter notebooks such as `biomedclip-image-classifier-latest.ipynb`.
2. **Run training** – Inside `image_only/github_push`, you will find a `configs/training_config.json` that can be passed to the training script:
   ```bash
   python -m src.train --config image_only/github_push/configs/training_config.json
   ```
3. **Inspect results** – Generated metrics, plots, and model checkpoints are stored under `image_only/github_push/output/`.
